# Sales and Customer Insights Dashboard using Power BI

## Overview

Developed an interactive BI dashboard using Power BI, Power Query, and DAX to analyze sales performance, customer behavior, top customers, and profitability metrics.

## Tools & Technologies

* Power BI
* Power Query
* DAX

## Features

* Sales Performance Analysis
* Customer Behavior Analysis
* Profitability Metrics
* Regional Trend Analysis
* Interactive Dashboard Reporting

## Insights Generated

* Identified regional sales trends and customer purchasing patterns.
* Analyzed top-performing customers and profitability metrics.
* Generated business insights to support data-driven decision-making.

## Outcome

Generated insights on regional trends to support business decisions using interactive dashboard visualizations and KPI reporting.
